# 🏋️ Fitness App - React Frontend Project

## 📌 תיאור הפרויקט

זהו פרויקט Frontend שנבנה ב־React כחלק ממודול פיתוח UI.
האפליקציה מדמה מערכת כושר אישית הכוללת עמודי אימונים, תזונה ומעקב התקדמות משתמש.

הפרויקט נבנה מתוך עיצוב שהוגדר מראש (Design System) וממומש לרכיבי React עם ניווט מלא בין עמודים.

---

## 🎯 מטרת המערכת

מטרת המערכת היא לאפשר למשתמש:
- לצפות בתוכניות אימון
- לקבל תוכנית תזונה יומית
- לעקוב אחרי התקדמות אישית
- להשתמש בממשק פשוט ונוח לשימוש

---

## 🧱 טכנולוגיות בשימוש

- React (Functional Components)
- React Router (ניווט בין עמודים)
- CSS Variables (Design System)
- CSS רספונסיבי
- Dummy Data בלבד (ללא Backend)

---

---

## 🌐 עמודים באפליקציה

### 🏠 Home Page (/)
עמוד הבית של האפליקציה עם ברוכים הבאים למערכת.

### 🔐 Login Page (/login)
עמוד התחברות בסיסי עם שדות אימייל וסיסמה.

### 🏋️ Workout Page (/workouts)
מציג רשימת אימונים זמינים (Cardio, Strength, Yoga).

### 🥗 Nutrition Page (/nutrition)
מציג תוכנית תזונה יומית וקלוריות.

### 📊 Progress Page (/progress)
מציג נתוני משתמש ומדדי גוף בסיסיים.

---

## 🎨 Design System

המערכת מבוססת על משתני CSS:

- Primary Color: `#4f46e5`
- Secondary Color: `#22c55e`
- Background: `#f9fafb`
- Text: `#111827`
- Border: `#e5e7eb`

העיצוב כולל:
- כרטיסים (Cards)
- כפתורים אחידים
- Navbar עליון
- התאמה למובייל (Responsive Design)

---

## 📱 רספונסיביות

המערכת מותאמת גם למסכים קטנים:
- Navbar משתנה לעמודה במובייל
- כרטיסים מתרחבים בהתאם לרוחב המסך
- שימוש ב־CSS Media Queries
