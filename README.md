# 🏋️ Fitness App - React Frontend Project

## 📌 תיאור הפרויקט

זהו פרויקט Frontend שנבנה ב־React כחלק ממודול פיתוח UI.
האפליקציה מדמה מערכת כושר אישית הכוללת עמודי אימונים, תזונה ומעקב התקדמות משתמש.

הפרויקט נבנה מתוך עיצוב שהוגדר מראש (Design System) וממומש לרכיבי React עם ניווט מלא בין עמודים.

---

## 🎯 מטרת המערכת

מטרת המערכת היא לאפשר למשתמש:
- לצפות בתוכניות אימון
- לקבל תוכנית תזונה יומית
- לעקוב אחרי התקדמות אישית
- להשתמש בממשק פשוט ונוח לשימוש

---

## 🧱 טכנולוגיות בשימוש

- React (Functional Components)
- React Router (ניווט בין עמודים)
- CSS Variables (Design System)
- CSS רספונסיבי
- Dummy Data בלבד (ללא Backend)
